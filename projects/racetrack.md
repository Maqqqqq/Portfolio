# Racetrack

Realtime operations dashboard for racetrack sessions, staff workflows, and public display views.

## Stack

- Node.js
- Express
- Socket.IO
- Sequelize
- SQLite

## Author

- Markus Stamm

## What It Does

- supports receptionist, race control, and safety roles
- tracks laps and race state in realtime
- exposes public-facing countdown and leaderboard views
- coordinates multiple screens around one running session
