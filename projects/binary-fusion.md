# Binary Fusion

Tool for combining two ELF binaries into a single executable that runs both.

## Stack

- Python
- GCC
- LIEF
- Linux ELF tooling

## What It Does

- accepts compiled ELF inputs
- builds a fused executable output
- demonstrates binary-format handling and runtime constraints

## Why It Belongs Here
