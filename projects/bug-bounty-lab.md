# Bug Bounty

School security-testing project against DVWA running on Metasploitable 2.

## Educational Use Only

This project documents a controlled lab exercise only. Do not apply these techniques to systems you do not own or do not have explicit permission to test.

## Author

- Markus Stamm

## Stack

- Linux
- DVWA
- Metasploitable 2
- VirtualBox

## What It Does

- documents scoped testing in a vulnerable lab
- captures findings and evidence
- includes remediation notes
- keeps testing boundaries explicit
