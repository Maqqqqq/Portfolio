# Port Prowler

CLI utility for TCP, UDP, and stealth-style port scanning.

## Stack

- Python

## Author

- Markus Stamm

## What It Does

- scans single ports, ranges, and lists
- supports TCP, UDP, and stealth probing modes
- formats output for terminal review or file export
- uses concurrency for larger scans
