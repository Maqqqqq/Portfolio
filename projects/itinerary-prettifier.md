# Itinerary Prettifier

Java CLI that turns raw itinerary text into more readable airport and time output.

## Stack

- Java
- file I/O
- CSV parsing
- date/time formatting

## What It Does

- reads itinerary input files
- expands airport codes from lookup data
- reformats date and time values
- writes cleaned output files

## Why It Belongs Here

- compact but clear parsing/transformation problem
- shows command-line Java beyond trivial exercises
