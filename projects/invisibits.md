# InvisiBits

Flask app for image and audio steganography with simple forensic inspection features.

## Stack

- Python
- Flask
- HTML/CSS
- Docker

## Author

- Markus Stamm

## What It Does

- hides text inside PNG, JPG, and WAV files
- decodes hidden payloads from uploaded media
- optionally locks payloads with password-derived protection
- estimates whether an image likely contains hidden content

## Why It Belongs Here

- distinct project with a clear domain
- combines backend logic with a usable interface
- strongest security-themed app in the set
