# Digital Detective

Python CLI for OSINT-style lookups against usernames, IP addresses, and people search data.

## Stack

- Python
- Typer
- requests
- external lookup APIs

## What It Does

- checks usernames across configurable platforms
- enriches IP addresses with provider and location data
- performs people-search style name lookups
- supports combined sequential searches in one run

## Why It Belongs Here

- practical command-line tool
- clear external integration work
- stronger than a toy script because it has a real user workflow
